# Ultrasound-Nerve-Segmentation
Kaggle competition on Deep Learning
This repository contains two deep learning-based medical image analysis projects:

Brain Tumor Detection – Automatically detects and classifies brain tumors from MRI scans using Convolutional Neural Networks (CNNs).
Ultrasound Nerve Segmentation – Segments nerve structures in ultrasound images using advanced image segmentation techniques such as U-Net to assist in medical diagnosis and treatment planning.

The objective of these projects is to leverage deep learning to improve the accuracy, efficiency, and reliability of medical image interpretation.

🎯 Features
Brain Tumor Detection
MRI image preprocessing and augmentation
Tumor detection and classification
CNN-based deep learning architecture
Model training and evaluation
Performance metrics visualization
Prediction on new MRI scans
Ultrasound Nerve Segmentation
Ultrasound image preprocessing
Pixel-wise nerve segmentation
U-Net architecture implementation
Mask generation and visualization
Segmentation performance evaluation
Prediction on unseen ultrasound images
